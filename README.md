# AI Snow Day Predictor

This project is an open-source tool for predicting snow days using the Mistral AI API and weather data.

## Setup

1. Clone this repository.
2. Open `index.html` in your web browser.
3. Enter your Mistral API key and location when prompted.

## Features

- Enter your Mistral API key, which is stored only in your current web session.
- Predict snow days based on your location and weather data.
- View a week-long prediction with each day in a different section.
- Navigate between days using the "Next Day" and "Go Back a Day" options.
