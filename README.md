# AI Snow Day Predictor

This project is an open-source tool for predicting snow days using the Mistral AI API.

## Setup

1. Clone this repository.
2. Open `index.html` in your web browser.
3. Enter your Mistral API key when prompted.

## Features

- Enter your Mistral API key, which is stored only in your current web session.
- Predict snow days based on your location and other factors.
